## 🎯 Cenários de Teste

### Fluxo Completo 1: Novo Usuário
1. Registrar conta (maria@example.com)
2. Completar perfil
3. Publicar primeira figurinha (Neymar)
4. Curtir outras figurinhas
5. Comentar em posts
6. Seguir usuários

### Fluxo Completo 2: Interação Social
1. Login (joao@example.com)
2. Ver feed
3. Curtir posts de Maria
4. Comentar
5. Seguir Maria
6. Enviar mensagem direta

### Fluxo Completo 3: Gestão de Coleção
1. Login (ana@example.com)
2. Adicionar 10+ figurinhas
3. Marcar diferentes status (tenho, quero, repetida)
4. Editar figurinha
5. Excluir figurinha
6. Filtrar por categoria

## 💡 Dicas para Testar

1. **Crie pelo menos 3 usuários** para simular interações sociais
2. **Publique 5-10 figurinhas** por usuário para popular o feed
3. **Faça interações cruzadas** (curtidas, comentários, follows)
4. **Teste o chat** entre usuários que se seguem mutuamente
5. **Experimente editar e excluir** suas próprias figurinhas
6. **Alterne entre modo claro e escuro** para ver o design
7. **Teste em mobile** (use DevTools → Toggle Device Toolbar)
